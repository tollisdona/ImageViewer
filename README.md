# ImageViewer
安卓实验作业：本地图片浏览器+爬虫
实现的功能：
1.加载本地图片文件夹，选择文件夹显示内部图片
2.输入网址爬取图片并展示

# BUG:
- 只能爬很旧的网站，没有改为js加载的，甚至好像用https协议的都不行。可以成功的例如中国蔬菜网。。。
- 页面逻辑有问题大概，点击文件夹跳转图片显示页面可以写个回调由Activity调用
